# 📊 ตารางเลเวล 1 - 110 (CookieRun Classic)

### 📌 นิยามและสูตรคำนวณพื้นฐาน (Formula Insights)
1. **EXP ที่ต้องใช้ (EXP required per level)**: จำนวน EXP ที่ต้องเก็บเพิ่มเพื่อผ่านเลเวลปัจจุบันไปเลเวลถัดไป
2. **EXP สะสม (Cumulative EXP)**: $\text{EXP สะสม(Lv } N) = \sum_{k=1}^{N} \text{EXP ต่อเลเวล}(k)$
3. **โบนัสเลเวล (%) (Level Bonus)**:
   - **เลเวล 1 – 50**: $\text{โบนัส} = \text{Level} - 1\,\%$
   - **เลเวล 51 – 110**: $\text{โบนัส} = 44 + (\text{Level} \times 0.1)\,\%$ (เช่น Lv 51 = 49.10%, Lv 110 = 55.00%)

## 4. ตัวอย่าง Code JS/TS Data Structure (สำหรับนำไปพัฒนาระบบคำนวณ)

```javascript
// ฟังก์ชันคำนวณโบนัสเปอร์เซ็นต์ตามเลเวล
function getLevelBonus(level) {
  if (level <= 50) return Math.max(0, level - 1);
  return Number((44 + (level * 0.1)).toFixed(2));
}

// ตัวอย่างฟังก์ชันคำนวณ EXP ที่ต้องใช้ระหว่างเลเวล A ถึง เลเวล B
function calculateRequiredExp(currentLevel, targetLevel, levelDataMap) {
  let totalExp = 0;
  for (let lv = currentLevel; lv < targetLevel; lv++) {
    const nextLevelData = levelDataMap[lv + 1];
    if (nextLevelData) {
      totalExp += nextLevelData.expRequired;
    }
  }
  return totalExp;
}
```
